About Petnames

Petnames are memorable, pronounceable names generated by combining random words. They're perfect for:

  • Server hostnames
  • Container names (Docker, Kubernetes)
  • Project codenames
  • Test environments
  • Anything that needs a friendly, memorable identifier

This tool is inspired by the original petname utility by Dustin Kirkland, used by Ubuntu and other projects.
